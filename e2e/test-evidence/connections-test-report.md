## UI 验证通过的功能

### ✅ 页面结构
- 页面标题正确显示"连接管理"
- 面包屑导航正确
- 侧边栏菜单正常

### ✅ 搜索区域
- 名称输入框（placeholder: "请输入连接名称"）
- 主机输入框（placeholder: "请输入主机地址"）
- 数据库输入框（placeholder: "请输入数据库名"）
- "查询"按钮
- "重置"按钮

### ✅ 添加连接对话框
- 标题："新增连接"
- 必填字段：连接名称、主机地址、端口、用户名、密码
- 可选字段：数据库名
- 连接池大小（默认10）
- "取消"和"确定"按钮

### ✅ 表格结构
- 列：ID、连接名称、主机地址、端口、用户名、数据库、状态、创建时间、操作
- 操作列按钮：编辑、测试、删除
- 空状态显示："暂无数据"

### ✅ 分页组件
- 显示"共 0 条"
- 每页条数选择（10条/页）
- 上一页/下一页按钮（禁用状态）
- 页码输入框
